This simple ad block defender plugin detects if a user is using ad blocking software and asks them to reconsider if they are.

== Description ==

If a reader is using ad blocking software, our plugin will prompt them with the following message every time they access one of your pages/posts: 

"This site uses advertisements to fund free content, please consider disabling your ad blocking software"

If the reader is not using ad blocking software, the plugin will not interface with the user. 

== Installation ==

1. Upload the `AdBlockDefender.zip` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Your site will automatically start prompting users to reconsider ad blocking software usage

== Frequently Asked Questions ==

= Can I change the message content and style that readers see if they are using ad block software? =

Future versions of this plugin will enable you to change the message and style of the alert box.
